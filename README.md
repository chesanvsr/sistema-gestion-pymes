# Sistema de Gestión para PYMEs

Aplicación fullstack desarrollada para la gestión de pequeñas y medianas empresas, permitiendo administrar usuarios, artículos, categorías e informes comerciales.

## 🚀 Funcionalidades

* Autenticación de usuarios mediante JWT
* Inicio de sesión con usuario y contraseña
* Gestión de usuarios
* Gestión de artículos
* Gestión de categorías
* Búsqueda de artículos por nombre y estado (activo/inactivo)
* Módulo de informes:

  * Ventas
  * Compras
  * Libro de IVA

## 🧠 Tecnologías utilizadas

### Backend

* Node.js
* Express
* Sequelize
* SQLite
* JWT (autenticación)

### Frontend

* React
* Vite
* Axios

## 📦 Estructura del proyecto

* `/backend`: API REST y lógica de negocio
* `/frontend`: interfaz de usuario

## ⚙️ Instalación y ejecución

### 1. Clonar repositorio

```
git clone https://github.com/chesanvsr/sistema-gestion-pymes.git
cd sistema-gestion-pymes
```

### 2. Backend

```
cd backend
npm install
npm run dev
```

### 3. Frontend

```
cd frontend
npm install
npm run dev
```
